Churn Prediction WebApp

Описание:
Веб-приложение для предсказания ухода клиентов (churn) на основе их характеристик. Пользователь может загружать CSV или TXT файл с данными клиентов, получать предсказания и видеть вероятность ухода прямо на веб-странице.

Возможности

Загружайте CSV или TXT файлы с данными клиентов

Получайте предсказание ухода (0 — останется, 1 — уйдёт)

Сразу отображается вероятность ухода

Высокий риск (вероятность > 0.5) подсвечивается красным

Результаты можно сохранить в CSV (с предсказаниями и вероятностями)

Красивый, интерактивный интерфейс на Bootstrap + JS

Технологии

Python 3.10+

FastAPI — backend и API

Pandas — обработка CSV/TXT

Scikit-learn — логистическая регрессия

Joblib — сохранение и загрузка модели

HTML/CSS/Bootstrap — оформление страницы

JavaScript — интерактивная загрузка файла и построение таблицы
